# **App Name**: TrackWise

## Core Features:

- Dashboard: Displays total income, expenses, and current balance.
- Add Expense Form: Form to input expense details (amount, category, description, date, type).
- Expense List: Displays a list of all expenses with an option to delete each.
- Add Expense: Allows users to add new expense records through the REST API.
- Delete Expense: Enables users to delete expense records using the REST API.
- Data Fetching: Fetches expense data from the backend REST API to update the dashboard and expense list.
- AI Expense Suggestion: AI tool that suggests expense categories based on transaction descriptions. Helps with faster categorization. (Needs REST endpoint for AI processing)

## Style Guidelines:

- Primary color: Soft blue (#79BAEC) for a calm and trustworthy feel.
- Background color: Light gray (#F5F7FA) for a clean and neutral backdrop.
- Accent color: Green (#90EE90) to highlight income and positive balances.
- Body and headline font: 'Inter', a grotesque-style sans-serif.
- Simple, outline-style icons to represent expense categories.
- Clean, single-page layout with clearly defined sections for dashboard, form, and expense list.
- Subtle transitions and animations when adding or deleting expenses.
